1.  Explain the differences between client-side routing and server-side routing.
Traditional server-side routing is when content requested by a page is pulled from a remote server. In client-side routing, the JavaScript handles this with what is already on the page (the client) and makes requests only for new or dynamic date.


2. What does HTTP stand for?
HTTP means HyperText Transfer Protocol.

3. What does CRUD stand for?
Create, Read, Update, Delete


4. Which HTTP methods can be mapped to the CRUD acronym that we use when interfacing with APIs/Servers.
Create - POST
Read - GET
Update - UPDATE
Delete - DELETE
